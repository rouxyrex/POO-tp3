//Compilation
Le makefile est inclus dans le repertoire et donc il suffit de taper 'make' dans le terminal pour compiler le programme

//Execution
Executer le programme avec './main'
Vous allez être présenté avec l'interface du programme
Il suffit de taper le chiffre correspondant à l'option recherché et appuyer sur la clé 'entrée' pour executer le service
NOTE: Les entrées texte (e.g. Nom de trajet, Ville de départ etc..) doivent être limitées à 3 characteres, sinon le programme peut avoir de problèmes de mémoires.
Utiliser l'option '6' pour quitter le programme

//Debugger
Utiliser le makefile 'makefile_debug' pour compiler le programme en sorte que les traces des constructeurs et destructeurs sont activés
