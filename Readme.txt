//Compilation
Le makefile est inclus dans le repertoire et donc il suffit de taper 'make' dans le terminal pour compiler le programme

//Execution
Executer le programme avec './main'
Vous allez être présenté avec l'interface du programme
Il suffit de taper le chiffre correspondant à l'option recherché et appuyer sur la clé 'entrée' pour executer le service
Utiliser l'option 'X' pour quitter le programme

//Debugger
Ajouter l'option '-D MAP' à CFLAGS dans le makefile pour compiler le programme en sorte que les traces des constructeurs et destructeurs sont activés
